
# How to Start

$ Install the python3 latest and pip

$ git clone https://github.com/Jayaraj8905/pytask.git or do download

$ cd pytask

# Install the project's dependencies
$ pip install flask

# Start the server1
$ cd server-1
$ python3 app.py

Server will run in below url
http://localhost:5001/

# Start the server2
$ cd server-2
$ python3 app.py

Server will run in below url
http://localhost:5002/

# To download the csv

Run http://localhost:5001/download
